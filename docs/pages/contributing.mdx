# Development

This React Use Hooks uses PNPM workspaces to manage the packages and Turbo Repo to manage the monorepo.

## Installation

```bash
pnpm install
```

## Commands

| Command      | Description                                                 |
|--------------|-------------------------------------------------------------|
| `pnpm build` | Build all packages.                                         |
| `pnpm dev`   | Start the development server for the documentation website. |
